# delicious_food
### 功能模块
- 菜谱列表展示 
- 搜索筛选菜谱
- 登录功能
- 投递和收藏

### 技术要点
- AngularJs单页应用
- AngualrJs过滤功能
- AngualrJs值验证
- json数据获取
- LESS渲染页面
- 自适应移动端
